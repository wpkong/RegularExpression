## 说明
* 该引擎支持 ?（可选）、*（零次或多次）、+（一次或多次）；
* 默认为非贪婪匹配模式，?只具备可选匹配功能；
* 支持范围[]，内部可以是选项[abc]，也可以是范围[a-zA-Z]；
* 支持多层括号分组，例如((ab|12)*|(xy|89)*)+ ；
* 支持通配符. ；
* 支持特殊字符: \d、\D、\w、\W、\s、\S；
* 支持转义字符，例如 ab\\(c 。

## 使用方法
```c 
#include <iostream>
#include "re.h"

int main(){
    RegularExpression re = "((ab|12)*|(xy|89)*)+";
    std::cout << re.match("12xy8912ab") << std::endl;
}
```
